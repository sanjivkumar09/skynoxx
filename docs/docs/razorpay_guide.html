💳 Razorpay Setup

Complete Integration Guide - Get Started in 5 Minutes

1
Create Razorpay Account

Sign up for a Razorpay account to get your API keys.

  1. Visit razorpay.com
  2. Click "Sign Up" → Create account with email
  3. Verify your email and phone number
  4. Login to dashboard.razorpay.com
Create Razorpay Account →
2
Get Your API Keys

Generate test API keys to start accepting payments.

  1. Login to Razorpay Dashboard
  2. Go to Settings → API Keys
  3. Click "Generate Test Keys"
  4. Copy your Key ID and Key Secret

⚠️ Security Warning

Never share your Key Secret publicly or commit it to Git. Keep it secure!

3
Configure Your Application

Update the configuration file with your Razorpay credentials.

File Location:

src/razorpay_config.php

Replace These Values:

// BEFORE (Default)
define('RAZORPAY_KEY_ID', 'rzp_test_YOUR_KEY_ID');
define('RAZORPAY_KEY_SECRET', 'YOUR_KEY_SECRET');

// AFTER (Your actual keys)
define('RAZORPAY_KEY_ID', 'rzp_test_AbCdEfGhIjKlMnOp');
define('RAZORPAY_KEY_SECRET', 'qRsTuVwXyZ1234567890');

📝 Configuration Checklist

  • Copied Key ID from Razorpay dashboard
  • Copied Key Secret (keep secure!)
  • Updated razorpay_config.php file
  • Set RAZORPAY_MODE to 'test'
  • Saved the file
4
Test the Integration

Use test credentials to verify payment flow works correctly.

🧪 Test Payment Credentials

success@razorpay

Testing Steps:

  1. Register a player account and join a tournament
  2. Go to Player Dashboard
  3. Click "Pay with Razorpay" button
  4. Click "Pay Now" on payment page
  5. Use test card/UPI credentials above
  6. Verify payment completes successfully
  7. Check dashboard shows "Completed" status
Go to Player Dashboard →
5
Go Live (Production)

When ready for real payments, switch to live mode.

⚠️ Prerequisites for Going Live

  • Complete KYC verification on Razorpay
  • Business account activated by Razorpay
  • Thorough testing in test mode completed
  • SSL certificate installed (HTTPS)

Live Mode Steps:

  1. Complete KYC in Razorpay dashboard
  2. Generate Live API Keys (starts with rzp_live_)
  3. Update razorpay_config.php with live keys
  4. Change mode: define('RAZORPAY_MODE', 'live');
  5. Test with small real transaction (₹1)
  6. Verify and refund test transaction
  7. Launch! 🚀
// Production Configuration
define('RAZORPAY_KEY_ID', 'rzp_live_YOUR_LIVE_KEY');
define('RAZORPAY_KEY_SECRET', 'your_live_secret');
define('RAZORPAY_MODE', 'live');
